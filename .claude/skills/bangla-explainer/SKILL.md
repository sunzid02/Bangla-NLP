# Bangla Explainer

**Purpose:** Explain each paper in easy, accessible Bangla.

**When to use:** At the end of every paper note (the `paper-extractor` output).

## Instructions

Write one short Bangla paragraph that answers:

1. **Problem** — এই পেপারটা কোন সমস্যা নিয়ে কাজ করেছে?
2. **Method/model** — কী method বা model ব্যবহার করেছে?
3. **Result** — কী result পেয়েছে?
4. **Limitation** — Limitation কী ছিল?
5. **Why useful for my thesis** — আমার thesis-এর জন্য কেন useful হতে পারে?

## Rules

- **3 to 5 simple Bangla sentences.**
- **Maximum 120 Bangla words.**
- Use simple, everyday language — **avoid jargon**.
- Write naturally; do not translate the paper title literally.
- Write in first person if it helps clarity ("এই গবেষণায়...").
