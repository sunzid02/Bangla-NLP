# Search Strategy

How to search for papers, batch by batch. (Inclusion/exclusion criteria live in
`docs/INCLUSION_EXCLUSION.md`; project rules in `AGENTS.md`.)

## Procedure

1. **Search ACL Anthology first.**
2. Search both keywords: **`Bangla`** and **`Bengali`**.
3. Filter years: **2024, 2025, 2026**.
4. Search **target venues first**: ACL, EMNLP, NAACL, COLING, EACL, AACL.
5. Then **cross-check** Google Scholar and Semantic Scholar for missing papers.
6. Keep a search log in **`logs/search-log.md`**.
7. Log rejected papers (with reason) in **`logs/rejected-papers.md`**.

## Token-Saving Rule

Do not expand every search result. First **collect candidates**, then verify only the
**shortlisted** papers.
